pick 0e40ba5 Licence
  • pick picorer le commit
  • reword picorer le commit, mais reformuler
  • edit picorer le commit en le modifiant
  • squash prendre le commit, mais le fusionner
  • fixup "squash", en éliminant le message
  • exec lancer la commande (reste de la ligne) dans un shell
  • drop supprimer le commit
